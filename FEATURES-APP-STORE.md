# App Store Release Roadmap

## Must-Have

### Polish UI/UX

- [ ] Interactive onboarding tour with feature discovery checklist
- [ ] Animations fluides, micro-interactions
- [ ] Icône app professionnelle
- [ ] Empty states engageants

### Features différenciantes

- [ ] Split view via drag and drop (like VSCode)
- [ ] Recherche full-text dans tous les fichiers
- [ ] Favoris / fichiers épinglés
- [ ] Historique des fichiers récents
- [ ] Tags / catégorisation

### Intégrations

- [ ] Sync iCloud / Dropbox
- [ ] Export PDF
- [ ] Quick Look extension (preview dans Finder)
- [ ] Spotlight indexing
- [ ] Share extension

### Personnalisation

- [ ] Thèmes (plusieurs palettes, pas juste dark)
- [ ] Choix de fonts (serif, sans-serif, mono)
- [ ] Custom CSS pour le rendu markdown

### Confiance

- [ ] Site web / landing page
- [ ] Privacy policy
- [ ] Support email
- [ ] Changelog visible dans l'app

## Nice-to-Have

- [ ] Mode présentation (plein écran, navigation slides)
- [ ] Mermaid diagrams rendering
- [ ] Math equations (LaTeX)
- [ ] Liens entre fichiers (wiki-style `[[links]]`)
- [ ] Mode focus (cache tout sauf le contenu)

## Bug Fixes / Improvements

- [ ] Apply settings live without reload (preserve open documents)
- [ ] Arrow keys scroll to next/previous section (based on TOC)

## UI Polish

### Quick Wins

- [ ] TOC panel (right) - style plus cohérent avec le reste
- [ ] Header document - afficher path complet au hover (tooltip)

### Améliorations

- [ ] Empty state sidebar quand aucun fichier trouvé
- [ ] Indicateur visuel (toast) quand on change font size / margin via raccourcis
- [ ] Hover states plus marqués sur la sidebar
- [ ] Animation de transition quand on switch de tab

## Pricing

Estimation : 5-15€ one-time ou 2-3€/mois
